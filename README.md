# NK Homes

Welcome to the NK Homes project repository.

## Project Overview
This website is built for NK Homes PVT Ltd, a premium home construction and interior design company.

## Tech Stack
- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Icons**: Lucide React

## Getting Started

1. **Install Dependencies**
   ```sh
   npm install
   ```

2. **Run Development Server**
   ```sh
   npm run dev
   ```

3. **Build for Production**
   ```sh
   npm run build
   ```

## Project Structure
- `src/components`: Reusable UI components
- `src/pages`: Route components
- `src/config`: Site-wide configuration (contact details, etc.)
- `src/lib`: Utility functions
